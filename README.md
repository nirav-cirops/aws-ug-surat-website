# AWS User Group Surat — Website

Official website for the [AWS User Group Surat](https://www.meetup.com/awsugsurat/) community.

## About

A static website for the AWS UG Surat community featuring:
- Upcoming & past event listings
- Community stats and organizer profiles
- Links to Meetup, Twitter, and LinkedIn

## Tech Stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- Vanilla JavaScript

## Project Structure

```
├── index.html        # Main website
├── style.css         # Custom styles
└── assets/
    ├── aws-logo.svg
    └── upcoming-event-nirav-mcp.png
```

## Running Locally

Just open `index.html` in your browser — no build step needed.
